OpenMediaVault 常用配置

这篇博文介绍了在 OpenMediaVault (OMV) 中启用 SSL 连接的步骤，以提高 Web 界面的安全性。用户需登录 OMV 管理界面，导航至 SSL/TLS 设置，申请或导入证书，启用 HTTPS 连接并保存更改。文中还详细说明了如何使用 Let's Encrypt 申请免费证书或导入自签名证书，以确保 SSL 连接的正常工作。

启用 SSL 连接

在 OpenMediaVault (OMV) 中，启用 SSL 连接可以提高访问 Web 界面的安全性，以下是启用 SSL 连接的步骤：

  1. 进入 OMV 管理界面：

    • 使用浏览器访问 OMV 的 IP 地址，例如 http://192.168.1.100。

  2. 登录 OMV 管理页面：

    • 使用默认的用户名 admin 和你设置的密码登录。

  3. 导航到 SSL/TLS 设置：

    • 点击左侧的 “系统” > “证书” > “SSL 证书”。

  4. 申请或导入证书：

    • 你可以选择自己申请证书，或者使用 Let’s Encrypt 申请免费证书。

  5. 启用 HTTPS 连接：

    • 进入 “服务” > “Web 管理”。
    • 勾选 “启用 HTTPS” 选项。
    • 选择刚刚申请或导入的证书。
    • 保存并应用更改。

  6. 测试 SSL 连接：

    • 使用 https:// 开头的 URL 重新访问 OMV 界面以确保 SSL 连接正常工作。

申请证书

  1. 使用 Let’s Encrypt 申请免费证书：

    • 确保你的 OMV 系统安装了 Let’s Encrypt 插件。
    • 在左侧导航中，选择 “服务” > “Let’s Encrypt”。
    • 填写你的域名、电子邮件，并按照提示完成申请。

  2. 导入自签名证书：

    • 如果不使用公共 CA 颁发的证书，可以生成自签名证书并导入。
    • 点击 “SSL 证书” > “添加”，选择 “自签名证书”，填写必要信息并生成证书。
