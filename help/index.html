关于/帮助

NeuroPix 是一个基于 Hugo 的静态博客项目，旨在提供一个简洁而优雅的平台，用于分享个人见解和创作✨。通过内置的 Makefile，用户可以轻松管理博客的生成、部署和文章创作📜，支持一键清理、生成静态文件以及将内容部署到 GitHub Pages🚀。此外，NeuroPix 允许用户快速创建新文章，并在本地服务器上实时预览更改，为写作提供了极大的便利与灵活性🔍。欢迎访问 NeuroPix，探索更多精彩内容！🌈

如何编辑

要编辑该博客，请首先运行以下命令克隆项目：

git clone --recurse-submodules https://github.com/NeuroPix/NeuroPix

注：NeuroPix 并非开源项目！它是基于 Hugo 和 Typo 主题构建的。

然后，按照项目地址的 README 文件进行编辑。
