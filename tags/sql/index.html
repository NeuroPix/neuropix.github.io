SQL 简明笔记

本文提供了数据库操作的基本SQL语法，包括数据定义、查询、更新和视图的创建。数据定义部分涵盖了模式、表和索引的创建、修改和删除。查询部分详细介绍了SELECT语句及其聚集函数、连接、ANY和ALL谓词、EXISTS谓词以及集合查询的用法。更新部分包括数据的插入、更新和删除操作。最后，视图的创建和删除也被简要提及，参考文献为王珊和萨师煊的《数据库系统概论》。
